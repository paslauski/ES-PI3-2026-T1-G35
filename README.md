# ES-PI3-2026-T1-G35

Aplicativo mobile para simulação de investimento em startups desenvolvido como parte do Projeto Integrador III do curso de Engenharia de Software da PUC-Campinas. 


1. Sobre o Projeto 

O MesclaInvest é uma aplicação mobile cujo objetivo é simular um ambiente digital de investimento em startups vinculadas ao ecossistema Mescla. 

A plataforma permite que usuários cadastrados acompanhem startups, visualizem informações institucionais e realizem negociações simuladas de tokens representativos dessas empresas. 

O sistema possibilita: 

Cadastro e autenticação de usuários, Visualização de startups cadastradas, Consulta de estrutura societária e dados simulados, Simulação de compra e venda de tokens, Controle de carteira com saldo fictício, Acompanhamento da valorização dos tokens por meio de dashboard gráfico.

O projeto possui caráter exclusivamente acadêmico e não envolve dinheiro real, blockchain real ou emissão de tokens reais. 

Na branch main, foi criada uma pasta data para armazenar arquivos de tabelas de startups nos formatos .xlsx e .csv, facilitando a organização e carregamento de dados simulados.
